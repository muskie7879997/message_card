【メッセージ追加方法】

このサイトは

messages/

フォルダ内の txt ファイルを読み込んで表示する。

ファイル名 = ID

例：

messages/R2010.txt
messages/R2011.txt
messages/R2012.txt

利用者が R2010 を入力すると

messages/R2010.txt

の内容が表示される。

【新しいメッセージを追加する手順】

1. txtファイルを作成
2. ファイル名を「ID.txt」にする
3. messagesフォルダへアップロード
4. Commit changes

例：

R2013.txt

内容：

おはよう。

今日は来てくれてありがとう。

これで利用者が R2013 を入力すると表示される。

【注意】

・ファイル名は大文字小文字を区別する
・拡張子は .txt
・文字コードは UTF-8 推奨
・index.html を編集する必要はなく、新しい txt を追加するだけでよい
